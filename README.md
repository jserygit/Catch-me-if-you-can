# Catch-me-if-you-can
2d game runner/bullet hell/rythmic qte

controls :
	Dash : R2/RT/A 
	
	Move : Z/S UP/DOWN
	
	Slow : L2/SHIFT
	
	Restart : R2/SPACE when you died
	
	Main Menu : START/TAB
	
	Pause : SHARE/ESC
